Sublime-memolist
================
I'm using the usual [memolist.vim](https://github.com/glidenote/memolist.vim).

and I want to use that with Sublime Text.


## Usage

Default:

* Dir `$HOME/memo`
* File `%Y-%m-%d-{title}.md`

Command | KeyMap
--- | --- 
Memo Open | `Command + M, Command + N`
Memo List | `Command + M, Command + L`
Memo Grep | `Command + M, Command + G`
